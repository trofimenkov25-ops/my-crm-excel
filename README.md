# 📊 CRM Система управления контактами в Excel

Полнофункциональная CRM система для управления контактами с базой данных на основе Excel файла.

## ✨ Возможности

✅ **Добавление контактов** - Создание новых контактов с полной информацией  
✅ **Поиск** - Быстрый поиск по имени, компании, email, телефону  
✅ **Фильтрация** - Фильтр по категориям и видам деятельности  
✅ **Редактирование** - Изменение информации о контактах  
✅ **Удаление** - Удаление ненужных контактов  
✅ **Статистика** - Анализ данных по категориям и видам деятельности  
✅ **Экспорт** - Выгрузка данных в CSV формат  
✅ **Автосохранение** - Все данные сохраняются в Excel файл  
✅ **Красивое оформление** - Профессиональное форматирование Excel файла  

## 📋 Поля контакта

- **ID** - Уникальный идентификатор
- **Категория** - Партнёр / Клиент / Поставщик / Конкурент
- **Вид деятельности** - Сфера деятельности компании
- **Название компании** - Полное название организации
- **ФИО** - Фамилия, имя, отчество контактного лица
- **Должность** - Должность в организации
- **Телефон** - Контактный номер
- **Email** - Электронная почта
- **WhatsApp** - Номер WhatsApp
- **Telegram** - Username в Telegram
- **Instagram** - Username в Instagram
- **Соцсети** - Ссылки на социальные сети
- **Сайт** - Веб-сайт организации
- **ИНН ЮЛ** - ИНН юридического лица
- **ЛПР** - Лицо, принимающее решение
- **Примечание** - Дополнительные заметки
- **Дата создания** - Автоматически записывается при добавлении
- **Дата обновлени��** - Обновляется при редактировании

## 🚀 Установка и запуск

### Требования
- Python 3.6 или выше
- pip (менеджер пакетов Python)

### Пошаговая инструкция

#### 1. Клонируйте репозиторий
```bash
git clone https://github.com/trofimenkov25-ops/my-crm-excel.git
cd my-crm-excel
```

#### 2. Создайте виртуальное окружение (опционально, но рекомендуется)
```bash
# На Windows
python -m venv venv
venv\Scripts\activate

# На macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

#### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

#### 4. Запустите программу
```bash
python crm.py
```

## ❓ Часто задаваемые вопросы

### Как открыть базу данных в Excel?
Просто откройте файл `CRM_Database.xlsx` обычным образом в Microsoft Excel или Google Sheets.

### Могу ли я редактировать данные прямо в Excel?
Да! Вы можете редактировать данные в Excel, и программа будет их читать при следующем запуске.

### Как восстановить удалённый контакт?
Удаленные контакты удаляются из Excel файла. Рекомендуется регулярно делать резервные копии файла.

### Может ли работать несколько пользователей одновременно?
Нет, программа работает с одним файлом Excel, поэтому одновременное редактирование может привести к конфликтам. Рекомендуется использовать по очереди или создать отдельные файлы для каждого пользователя.
